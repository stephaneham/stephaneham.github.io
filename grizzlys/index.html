Grizzlys

Grizzlys is the professional esports team for Montreal Esports. Here are the things that I've done with Grizzlys;

  • Developing and deploying a Wordpress theme from scratch with the help of a designer using HTML/CSS/Javascript/JQuery/PHP.
  • Building minor plugins.

Accomplishments

  • Developed the theme under the short time frame given.
  • Integrated the WPML plugin into the theme.

Features of Interest

Fully Responsive Theme

The Grizzlys Wordpress theme has been built to be viewable both on mobile and desktop devices.

WPML Integrated

The Grizzlys' theme has WPML integrated into it, allowing for translations in different languages.

Plugins

The Grizzlys' theme has two custom shortcode plugins for Google Forms and for the Toornament platform.
