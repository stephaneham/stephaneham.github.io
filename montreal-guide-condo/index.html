Montreal Guide Condo

During my time with Montreal Guide Condo, I rebuilt their web platform as their old installation did not meet their needs. This includes;

  • Modeling a relational MySQL Database to hold data about the listings, advertisements, areas etc...
  • Building a MVC backend in PHP for both the Montreal Guide Condo website and their content management system.
  • Building the frontend using HTML5/CSS3/Javascript/JQuery for the Montreal Guide Condo website and the content management system.
  • Analyzing user behavior on the website.
  • Handling SEO in the frontend and backend.

Accomplishments

  • Rebuilt an obsolete platform, leading to better usability for both visitors and employees.
  • Drastically decreasing the load time of pages, leading to user satisfaction, especially on mobile devices.
  • Improving the SEO of the website, making it triple its organic visitors per month over a year.

Features of Interest

Fully Responsive Website

MtlGC has a fully responsive interface built in CSS, allowing visitors to browse the website at their leisure on any device.

Improved Search Features

The old platform had a fairly rudimentary search feature. The new one bolsters multiple options to select from, as well as an autocomplete feature that uses AJAX to list regions and cities depending on their popularity.

The new system also included an implementation of clusters over Google Maps, as well as a pagination system, which allows users to load extra pages of listings with the press of a button using AJAX calls.

Proximity Features

A MySQL algorithm allows the platform to showcase real estate projects close to the one in the listing viewed.

This same algorithm was later re-used to show which real estate projects are close to metro stations in Montreal.

Facebook API Implementation

Users can log onto the website to use the "favorites" feature, which allows users to store listings in their "favorites" section. One of the ways they can log in is with Facebook. This implementation uses the PHP Facebook SDK.

Paypal API Implementation

Advertisers can add their advertisements on the MtlGC website through an interface that uses the PHP Paypal SDK to communicate with Paypal.

A Custom Content Management System

The custom content management system is used to manage listings for both Montreal Guide Condo and Ma Residence Retraite.

You can browse all the listings and search for specific listings to edit using the search bar.

Within the add/edit screen, you can change the many properties of a listing. A feature fetches information from Wordpress posts from the Montreal Guide Condo blog to store them on the Montreal Guide Condo website.

A file upload system, as well as an implementation of Cropper ( https://fengyuanchen.github.io/cropper/ ) and JQuery UI's Sortable ( https://jqueryui.com/sortable/ ) allows the user to upload and crop images for a listing. Once cropped, the image is converted to many formats by PHP using the Imagick library.
