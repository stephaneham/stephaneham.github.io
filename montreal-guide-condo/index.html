About Montreal Guide Condo / Ma Residence Retraite

Montreal Guide Condo is an online directory primarly for condos in the city of Montreal and the surrounding areas. It aims to help users in their search for a condo by providing them with many tools, including a search engine, map features, and an extensive page for each real estate project.

Ma Residence Retraite is very similar in that regard, except it specializes in retirement homes for the elderly. It functions off the same system at Montreal Guide Condo

Features of Interest

Fully Responsive Website

Montreal Guide Condo has a fully responsive interface built in HTML/CSS, allowing visitors to browse the website at their leisure on any device.

Improved Search Features

The old platform had a fairly rudimentary search feature. The new one bolsters multiple options to select from, as well as an autocomplete feature that uses AJAX to list regions and cities depending on their popularity.

Improved Navigation

The new system also included an implementation of clusters over Google Maps, as well as a pagination system, which allows users to load extra pages of listings with the press of a button using AJAX calls.

Proximity Features

A MySQL algorithm allows the platform to showcase real estate projects close to the one in the listing viewed.

Condos Close to a Metro Station

The proximity algorithm was later re-used to show which real estate projects are close to metro stations in Montreal.

Facebook API Implementation

Users can log onto the website to use the "favorites" feature, which allows users to store listings in their "favorites" section. One of the ways they can log in is with Facebook. This implementation uses the PHP Facebook SDK.

Paypal API Implementation

Advertisers can add their advertisements on the Montreal Guide Condo website through an interface that uses the PHP Paypal SDK to communicate with Paypal.

A Custom Content Management System

The custom content management system is used to manage listings for both Montreal Guide Condo and Ma Residence Retraite.

CMS Listing Browsing

You can browse all the listings and search for specific listings to edit using the search bar.

CMS Adding and Editing Listings

Within the add/edit screen, you can change the many properties of a listing. A feature fetches information from Wordpress posts from the Montreal Guide Condo blog to store them on the Montreal Guide Condo website.

CMS Cropping Images

A file upload system, as well as an implementation of Cropper and JQuery UI's Sortable allows the user to upload and crop images for a listing. Once cropped, the image is converted to many formats by PHP using the Imagick library.
