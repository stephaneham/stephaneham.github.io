Montreal Guide Condo

During my time with Montreal Guide Condo, I rebuilt their web platform as their old installation did not meet their needs. This includes;

  • Modeling a relational MySQL Database to hold data about the listings, advertisements, areas etc...
  • Built a MVC backend in PHP for both the Montreal Guide Condo website and their content management system.
  • Built the frontend using HTML5/CSS3/Javascript/JQuery for the Montreal Guide Condo website and the content management system.
  • Analyzing user behavior on the website.
  • Handling SEO in the frontend and backend.

Accomplishments

  • Rebuilt an obsolete platform, leading to better usability for both visitors and employees.
  • Drastically decreasing the load time of pages, leading to user satisfaction, especially on mobile devices.
  • Improving the SEO of the website, making it triple its organic visitors per month over a year.

Features of Interest

Fully Responsive Website

MtlGC has a fully responsive interface built in CSS, allowing visitors to browse the website at their leisure on any device.

Improved Search Features

The old platform had a fairly rudimentary search feature. The new one bolsters multiple options to select from, as well as an autocomplete feature that uses AJAX to list regions and cities depending on their popularity.

Proximity Features

An algorithm allows the platform to showcase homes close to the one in the listing viewed.
