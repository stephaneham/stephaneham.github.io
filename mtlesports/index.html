Montreal Esports

Montreal Esports is a startup that delves into "Electronic Sports", otherwise known as esports. Esports are video game competitions played by professional athletes. During my time at Montreal Esports, I set up their tech infrastructure and worked on some websites. As this is the Montreal Esports page, I will only talk about the infrastructure and the Montreal Esports website. If you wish to see the other websites, check out the Grizzlys and ESNC pages. Here are the things that I've done at Montreal Esports;

  • Checking on local dedicated server providers and renting an appropriate server.
  • Setting up the server under a Linux Operating System.
  • Managing the DNS for 7 different domains using Cloudflare.
  • Installing and managing a mail server using a Postfix/Dovecot instance with DKIM authentication and a SpamAssassin filter.
  • Installing and managing a LAMP (Linux, Apache, MariaDB/MySQL, PHP) stack.
  • Installing and managing a caching service (Redis).
  • Setting up DDoS protections using Cloudflare and Amazon Servers.
  • Deploying a Wordpress solution for the MtlEsports service.

Accomplishments

  • Gave a fledging startup the infrastructure it needs to operate online.
  • Found a quick solution to deliver the MtlEsports website in the requested timeframe.
  • Delved into a lot of different services.
