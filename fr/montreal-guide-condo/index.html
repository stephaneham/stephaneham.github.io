Montreal Guide Condo / Ma Résidence Retraite

Montreal Guide Condo est un répertoire en ligne principalement pour les condos dans la ville de Montréal et les environs. Son objectif est d'aider les usagers dans leur recherche d'un condo en leur fournissant divers outils, incluant un engin de recherche, des fonctionalités de carte, et une page extensive pour chaque projet immobilier sur le site.

Ma Résidence Retraite est un site très similaire fonctionnant sur le même système que Montreal Guide Condo. Cependant, il se spécialise dans les habitations pour les gens à la retraite.

Caractéristiques d'Intêret

Site Web 100% "responsive"

Montreal Guide Condo a une interface 100% "responsive" construite en HTML/CSS, permettant aux visiteurs de naviguer le site web à leur convenance sur n'importe quel appareil.

Fonctionalités de Recherche Améliorées

L'ancienne plateforme avait un engin de recherche rudimentaire. La nouvelle a un engin fournissant plusieurs choix afin de préciser sa recherche, ainsi qu'une fonctionalité "autocomplete" qui utilise AJAX pour afficher des régions et des villes selon leur popularité.

Navigation Améliorée

Le nouveau système inclus une implémentation des groupages sur Google Maps, ainsi qu'un système de pagination qui permet aux usagers des charger les pages suivantes au clic d'un bouton utilisant des appels AJAX.

Détection de Proximité

Un algorithme MySQL permet à la plateforme de montrer des projets immobiliers près du projet en visionnement.

Condos Près d'une Station de Métro

L'algorithme de proximité a été ré-utilisé pour montrer les projets immobilier à proximité des stations de métro de Montréal.

Implémentation API Facebook

Les usagers peuvent se connecter sur le site web pour utiliser la fonctionalité "favoris", ce qui permet à l'usager de sauvegarder leurs projets favoris pour un visionnement ultérieur. Une des façons de se connecter est avec Facebook. Cette fonctionalité utilise le SDK PHP de Facebook.

Implémentation API Paypal

Les annonceurs peuvent acheter et placer leurs publicités sur Montreal Guide Condo à partir d'une interface qui utilise le PHP SDK Paypal pour communiquer avec Paypal.

Gestionnaire de Contenu sur Mesure

Le gestionnaire de contenu est utilisé pour gérer les listings pour Montreal Guide Condo et Ma Résidence Retraite.

Navigation de Listings dans le Gestionnaire de Contenu

On peut naviguer au travers de tous les listings et chercher pour un listing spécifique en utilisant une barre de recherche.

Ajout et Édition de Listings

Dans l'interface d'ajout/édition, on peut changer les propriétés d'un projet. Une fonctionnalité permet d'obtenir de l'information des articles du blog Wordpress de Montreal Guide Condo pour les afficher sur leur site.

Découpage d'Image

Un système de téléversement, ainsi qu'une implémentation de Cropper et de JQuery UI Sortable permettent à l'usager de téléverser et découper une image pour un listing. Une fois découpée, l'image est convertie dans plusieurs format par PHP en utilisant la librairies Imagick.
