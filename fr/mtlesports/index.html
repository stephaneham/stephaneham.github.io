Montreal Esports

Montreal Esports est une startup qui plonge dans le "Sport Électronique", autrement connu comme "esports". Ils visent à supporter les joueurs professionel en esports en leur fournissant du support, une agence de joueur, et une chaîne de contenu esports.

CARACTÉRISTIQUES D'INTÊRET

Thème Wordpress Intégré

Montreal Esports utilise le thème Wordpress Fluid pour son site web.

Integration of the WPML Plugin

Intégration du plugin WPML

Montreal Esports uses WPML to allow users to view the website in French.

Integration of the Slider Revolution Plugin

Montreal Esports uses Slider Revolution onf its frontpage for a parallax effect.

Integration of a Calendar Plugin

Montreal Esports uses the Simple Calendar in the Calendar page to showcase its upcoming events.

Redis Cache

Montreal Esports uses Redis for its object caching, installed on the dedicated server, as well as the WP Redis plugin.

Services Provided

During my time with Montreal Esports, I've provided support in many aspects, such as;

  • Checking on local dedicated server providers and renting an appropriate server.
  • Setting up the server under a Linux Operating System.
  • Managing the DNS for 7 different domains using Cloudflare.
  • Installing and managing a mail server using a Postfix/Dovecot instance with DKIM authentication and a SpamAssassin filter.
  • Installing and managing a LAMP (Linux, Apache, MariaDB/MySQL, PHP) stack.
  • Installing and managing a caching service (Redis).
  • Setting up DDoS protections using Cloudflare and Amazon Servers.
  • Deploying a Wordpress solution for the MtlEsports website.
